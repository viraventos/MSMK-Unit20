# Ejemplos HolaMundo en JS y TS

## Ejemplo HolaMundo en TypeScript

1. Crear Fichero *holamundo.ts* con el código que queremos ejecutar.
2. Ejecutar el código: `npx tsx holamundo.ts`
3. Ejecutar el código: `node holamundo.ts` (Esto no siempre funciona)

## Ejemplo HolaMundo en Javascript

1. Crear Fichero *holamundo.js* con el código que queremos ejecutar.
2. Ejecutar el código: `node holamundo.js`
